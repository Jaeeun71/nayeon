## 📖 NayeonWiki-Birthday (나연 위키)
나무위키의 문서 구조와 스타일을 재현하여, 

소중한 친구의 정보를 사전 형식으로 기록하고 생일을 축하하는 웹 프로젝트입니다.

### 🔗 Project Overview

컨셉: 나무위키 '3월 11일' 문서 및 인물 정보 문서 클론 코딩

제작 기간: 2024.03 (생일 기념)

주요 특징: - 실제 나무위키의 레이아웃(목차, 인물 정보 박스, 최근 수정 시각 등) 재현

여러 HTML 파일을 하이퍼링크로 연결한 다중 페이지 구조

### 🛠 Tech Stack
Frontend: HTML5, CSS3

Design System: NamuWiki UI 스타일 가이드 참조 (Blue-colored links, Gray borders)


### 🤖 AI Collaboration Experience
이 프로젝트는 AI(ChatGPT)와 페어 프로그래밍을 통해 효율적으로 제작되었습니다.

UI 패턴 분석: 나무위키 특유의 복잡한 문서 구조(인물 정보 테이블, 다중 계층 목차)를 HTML로 빠르게 변환하기 위해 AI의 코드 생성을 활용했습니다.

CSS 디테일 구현: visited 링크의 색상 변화나 인라인 요소(display: inline) 처리 등 위키 특유의 가독성을 높이는 스타일링 작업을 AI와 함께 검토하며 최적화했습니다.

생산성 향상: 반복적인 HTML 태그 작성 시간을 단축함으로써, 친구를 위한 맞춤형 콘텐츠 기획 및 텍스트 작성에 더 많은 시간을 할애할 수 있었습니다.
